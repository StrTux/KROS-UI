## [0.1.0] — 2025

### Initial Open-Source Release 🎉

**KROS UI** — React Native CLI component library for iOS & Android.

### Added Components (~35 complete)
- **Accordion** — Collapsible content sections
- **Alert** — Informational banners with variants
- **Alert Dialog** — Modal confirmation dialogs
- **AI Input** — Animated AI-style text input
- **Article** — Rich article/blog card layout
- **Aspect Ratio** — Fixed aspect ratio container
- **Avatar** — User avatar with fallback
- **Badge** — Status and label badges
- **Breadcrumb** — Navigation breadcrumb trail
- **Button** — Full-featured button (6 variants, 6 sizes, icons, loading, disabled)
- **Button Group** — Grouped button layouts
- **Calendar** — Interactive date calendar
- **Card** — Flexible content card with sub-components
- **Carousel** — Swipeable image/content slider
- **Chart** — Basic charts (bar, line)
- **Checkbox** — Accessible checkbox input
- **Data Table** — Sortable, paginated data table
- **Date Picker** — Date selection component
- **Dialog** — Accessible modal dialog
- **Drawer** — Bottom/side slide-out drawer
- **Dropdown Menu** — Context menus with radio & checkbox items
- **Empty** — Empty state placeholder
- **Field** — Form field wrapper with label/error
- **Form** — Complete login & signup forms
- **Hover Card** — Hover-activated preview card
- **Input** — Text input with variants
- **Input Group** — Input with prefix/suffix addons
- **Input OTP** — OTP/PIN digit input group
- **Item** — List item with media, content, actions
- **Label** — Accessible form label
- **Menubar** — Horizontal navigation menubar
- **Music Player** — Full-featured audio player card
- **Popover** — Floating content popover
- **Profile Card** — User profile display card
- **Progress** — Linear progress indicator
- **Purchase Card** — E-commerce purchase card
- **Radio Group** — Single-select radio buttons
- **Spinner** — Loading spinners (multiple variants)
- **Streaming** — Live streaming card UI
- **Switch** — Toggle switch
- **Table** — Structured data table
- **Tabs** — Tab navigation
- **Text** — Typography component
- **Textarea** — Multiline text input

### Core System
- `applyTw()` — Tailwind CSS utility class engine for React Native
- Poppins font integration (18 variants — all weights + italics)
- Dark/Light theme context system (`ThemeProvider`, `useTheme`)
- Simple screen router (no React Navigation dependency)
- Flaticon icon font integration

### Project Setup
- React Native 0.74.3
- React Native Reanimated 3.10.x
- react-native-safe-area-context
- react-native-linear-gradient
- react-native-svg
- react-native-video
- class-variance-authority
